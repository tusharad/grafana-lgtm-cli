## Adding gemini ai

Add a layer that will handle the LLM calling and receving response part.

Primary goal is to make two types of LLM calls.

1. One for generating a prometheus query based on natural language request.

## Input
    
    Prompt (text) containing user question (I want to know highest number of request per second in last 3 hours)

## Output

    A structured json response contiaining the query


2. One for summarizing the prometheus query result.

## Input
    
    Prompt (text) containing query result 

## Output

    meaning of this result

Use Gemini client
